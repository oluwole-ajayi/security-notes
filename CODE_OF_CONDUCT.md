# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: everyone participating in this repository — through issues, pull requests, or discussions — is expected to treat others with respect and good faith. No harassment, no discrimination, no bad-faith engagement.

Instances of unacceptable behaviour can be reported by contacting me via [LinkedIn](https://www.linkedin.com/in/ajayi-isaac/) or through GitHub's own reporting channels.

Enforcement is at my discretion.
